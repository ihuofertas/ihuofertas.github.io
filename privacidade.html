Política de Privacidade

Última atualização:16/10/2025.

Esta Política de Privacidade descreve como o aplicativo AVG3 (o "Aplicativo") lida com as informações ao interagir com a API da plataforma Pinterest. O uso deste Aplicativo é destinado a fins pessoais e/GAE ou organizacionais internos, conforme descrito no formulário de desenvolvedor do Pinterest.

1. Coleta e Uso de Informações O Aplicativo NÃO coleta, armazena, vende ou compartilha quaisquer dados pessoais identificáveis dos seus usuários (como nome, e-mail ou informações de perfil). O Aplicativo acessa a API do Pinterest unicamente com as permissões concedidas pelo usuário (via OAuth) e apenas para as finalidades estritamente necessárias ao seu funcionamento.

2. Finalidade do Aplicativo e Uso de Dados A finalidade deste Aplicativo é permitir ao proprietário da conta enviar e agendar "Pins" para sua própria conta do Pinterest; gerar relatórios da própria conta do Pinterest; e gerenciar campanhas de anúncios da própria conta do Pinterest. Para isso, o Aplicativo utiliza as permissões da API para: * Autenticar o Usuário: Usamos o token de acesso fornecido pelo Pinterest para verificar a identidade do usuário e autorizar ações na plataforma. * Criar e Agendar Pins: O Aplicativo envia os dados fornecidos pelo usuário (imagens, links de destino, títulos e descrições) diretamente para a API do Pinterest para criar ou agendar a publicação de novos Pins. * Gerar Relatórios Pessoais: O Aplicativo poderá obter dados de sua própria conta do Pinterest. * Gerenciar Campanhas de Anúncios: O Aplicativo poderá gerir campanhas e anúncios da própria conta do Pinterest.

3. Armazenamento de Dados Nenhum dado pessoal, informação de perfil ou conteúdo de Pin é armazenado em bancos de dados ou servidores pertencentes ou controlados por este Aplicativo. Os tokens de autenticação (OAuth tokens) necessários para a comunicação com a API podem ser armazenados localmente no ambiente do usuário (por exemplo, em um arquivo de configuração local ou variável de ambiente) para manter a sessão ativa. Esses tokens não são transmitidos ou acessíveis por nós.

4. Segurança O Aplicativo utiliza as práticas de segurança padrão (como conexões HTTPS) para se comunicar com a API do Pinterest, garantindo que os dados em trânsito sejam criptografados.

5. Alterações nesta Política Esta Política de Privacidade pode ser atualizada ocasionalmente. Notificaremos sobre quaisquer alterações publicando a nova política nesta página e atualizando a data da "Última atualização"
