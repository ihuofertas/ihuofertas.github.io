Política de Privacidade

Última atualização:22/10/2025.

Esta Política de Privacidade descreve como o aplicativo IhuAutomatV2 lida com as informações ao interagir com a API da empresa META e relacionados. O uso deste Aplicativo é destinado a fins pessoais e/GAE ou organizacionais internos, e não possui usuários além do próprio administrador.

1. Coleta e Uso de Informações
O Aplicativo NÃO coleta, armazena, vende ou compartilha quaisquer dados pessoais identificáveis dos seus usuários (como nome, e-mail ou informações de perfil). Para seu funcionamento, o Aplicativo acessa apenas as seguintes informações, com a permissão explícita do administrador:

  • Tokens de Acesso de API: Chaves de autenticação (tokens) para as APIs do Facebook e Instagram, permitindo que o aplicativo publique conteúdo em nome do administrador.
  • Dados de Planilha Google: O aplicativo lê dados de uma planilha Google Sheets privada, pertencente ao administrador, para criar as postagens.

2. Finalidade do Aplicativo e Uso de Dados
A finalidade deste Aplicativo é permitir ao proprietário da conta enviar e agendar postagens para sua própria conta do Facebook e Instagram; gerar relatórios da própria conta; e gerenciar campanhas de anúncios da própria conta.

3. Exclusão de Dados e Revogação de Acesso para usuários
Como o Aplicativo não armazena dados de usuários, a "exclusão de dados" refere-se à revogação do acesso do Aplicativo às suas contas de rede social. O administrador pode revogar o acesso do Aplicativo a qualquer momento seguindo estes passos:
Para Facebook e Instagram:

  1. Acesse sua conta do Facebook.
  2. Vá para "Configurações e Privacidade" > "Configurações".
  3. No menu lateral, clique em "Aplicativos e sites" (ou "Integrações comerciais").
  4. Encontre o aplicativo "IhuAutomatV2" na lista.
  5. Clique em "Remover" para revogar todas as permissões.

4. Armazenamento de Dados
Nenhum dado pessoal, informação de perfil ou conteúdo é armazenado em bancos de dados ou servidores pertencentes ou controlados por este Aplicativo. Os tokens de autenticação necessários para a comunicação com a API podem ser armazenados localmente no ambiente do usuário (por exemplo, em um arquivo de configuração local ou variável de ambiente) para manter a sessão ativa. Esses tokens não são transmitidos por nós.

5. Segurança
O Aplicativo utiliza as práticas de segurança padrão (como conexões HTTPS) para se comunicar com a API, garantindo que os dados em trânsito sejam criptografados.

6. Alterações nesta Política
Esta Política de Privacidade pode ser atualizada ocasionalmente. Notificaremos sobre quaisquer alterações publicando a nova política nesta página e atualizando a data da "Última atualização"
